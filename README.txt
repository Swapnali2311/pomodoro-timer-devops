Pomodoro Timer - Small Project
------------------------------

How to use:
1. Unzip the folder and open `index.html` in your browser.
2. Adjust Work / Short Break / Long Break lengths if you like.
3. Click Start to begin the timer. Use Pause and Reset as needed.

Files included:
- index.html
- style.css
- script.js
- README.txt

This is a minimal, plain HTML/CSS/JS project meant for learning and quick customization.
